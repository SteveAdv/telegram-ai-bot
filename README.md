# Telegram AI Bot

## Setup Instructions
1. **Clone the repository**:  
   Run the following command in your terminal:  
   ```bash  
   git clone https://github.com/SteveAdv/telegram-ai-bot.git  
   ```  

2. **Navigate to the project directory**:  
   ```bash  
   cd telegram-ai-bot  
   ```  

3. **Install dependencies**:  
   Make sure you have [Node.js](https://nodejs.org/) installed. Then, run:  
   ```bash  
   npm install  
   ```  

## Usage Guide
1. **Create a .env file**:  
   Create a `.env` file in the root directory and add your API tokens. Example:  
   ```
   TELEGRAM_BOT_TOKEN=your_telegram_bot_token
   OPENAI_API_KEY=your_openai_api_key
   ```  

2. **Run the bot**:  
   Use the following command:  
   ```bash  
   npm start  
   ```  
   Your bot should now be running!

## Deployment Steps for Railway
1. **Sign in to Railway**:  
   Go to [Railway](https://railway.app/) and create an account or log in.

2. **Create a new project**:  
   Click on `New Project` and select `Deploy from GitHub`.  

3. **Connect your GitHub account**:  
   Authorize Railway access to your GitHub account.  

4. **Select the repository**:  
   Choose `telegram-ai-bot` from your repositories.

5. **Set up environment variables**:  
   Add the required environment variables as specified in your `.env` file in Railway's settings.

6. **Deploy**:  
   Click `Deploy` and wait for the deployment to complete. Your bot will be live soon!

- **Usage Logging**:  
  All interactions are logged for monitoring.
